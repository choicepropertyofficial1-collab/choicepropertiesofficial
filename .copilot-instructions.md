## 📊 PROJECT TYPE

This is a **PURE STATIC WEBSITE**:
- Frontend: Vanilla HTML, CSS, JavaScript
- Backend: Supabase cloud (Edge Functions + PostgreSQL)
- Deployment: Cloudflare Pages (static files only)
- Build: `node generate-config.js` (runs only at deploy time)
- Runtime: Zero dependencies (no npm packages at runtime)

---

## 🏗️ Architecture (CRITICAL UNDERSTANDING)

```
User's Browser
    ↓ (Static HTML/CSS/JS)
Cloudflare Pages
    ↓
Azure CDN (cached)
    ↓
User sees website

When user interacts:
    ↓ (JavaScript)
Browser calls Supabase Edge Functions
    ↓ (Server-side logic)
Supabase Edge Functions → Database (PostgreSQL)
    ↓ (Results)
Response back to browser
```

**KEY POINT:** This repository contains ONLY the browser layer (static files).
The backend is entirely on Supabase (separate from this repo).
